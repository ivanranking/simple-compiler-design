SimpleScript Language Guide

Data Types

int, float, string, boolean

Keywords

var, function, if, else, for, while, return, print, input, true, false

Operators

+, -, *, /, %, ==, !=, <, >, <=, >=, and, or, not

Example

var x = 10;
var y = 20;
if (x < y) {
    print("x is less than y");
}

function greet(name) {
    print("Hello, ", name);
}
greet("World");
