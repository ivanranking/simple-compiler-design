## 📋 Compilation Process

Every time you compile:

### Phase 1: Lexical Analysis
- Source code → Tokens
- See in: "Tokens" tab
- Example: `var x = 10;` → [VAR, IDENTIFIER, ASSIGN, INTEGER, SEMICOLON]

### Phase 2: Parsing
- Tokens → Abstract Syntax Tree
- See in: "AST" tab
- Shows hierarchical program structure

### Phase 3: Semantic Analysis
- Type checking and validation
- Happens behind the scenes
- Errors shown in "Errors" tab

### Phase 4: Execution
- AST interpretation
- Statement and expression evaluation
- Output shown in "Output" tab
